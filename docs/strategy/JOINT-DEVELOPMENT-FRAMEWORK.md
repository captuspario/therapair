1. **Technical Testing:** Prequel conducts technical testing
2. **Clinical Testing:** Unison conducts clinical testing
3. **User Testing:** Joint user testing and feedback
4. **Quality Assurance:** Joint quality assurance and validation

2. **Clinical Validation:** Unison validates clinical functionality

4. **User Training:** Unison provides clinical user training
